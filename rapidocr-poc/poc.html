RapidOCR POC

Load a handle photo, rotate if needed, pan/zoom, then run OCR. Boxes are colored by confidence: green ≥0.9, yellow ≥0.5, red below; gray dashed = drawn but not yet recognized.

Mouse / gesture / key shortcuts
Left-drag on empty canvasDraw a new box
Click a boxSelect it (click again to deselect)
Ctrl + left-dragPan
Two-finger scrollPan
PinchZoom (anchored at cursor)
Delete / BackspaceRemove selected box
Hover near a box's top-right corner, click the ×Delete that box immediately
Click a result in the listSelect/deselect its box
