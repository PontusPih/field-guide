Scan

Load a handle photo, rotate if needed, pan/zoom, then run OCR. Boxes are colored by confidence: green ≥0.9, yellow ≥0.5, red below; gray dashed = drawn but not yet recognized; dark red dashed = recognition tried and found nothing. Each box shows just a number by default — hover or select it (or its row in the list) to see its full text.

Mouse / gesture / key shortcuts
Hover a boxReveal its full text (default view just shows a number)
Left-drag on empty canvasDraw a new box
Click a boxSelect it (click again on its body to deselect)
Drag a selected box's bodyMove it
Drag a selected box's corner handleResize it (editing resets it to unrecognized)
Ctrl + left-dragPan
Two-finger scrollPan
PinchZoom (anchored at cursor)
Delete / BackspaceRemove selected box
Hover near a box (or select it), click the × floating above itDelete that box immediately
Click a result in the listSelect/deselect its box
🔍 / × next to a list rowPan/zoom to that box / delete it directly
